Bubble POP-UP

서울시 실제 데이터를 활용한 팝업스토어 경영 시뮬레이션 게임입니다. 서울 8개 상권의 유동인구, 임대료, 교통, 뉴스 데이터를 기반으로 지역을 선택하고, 메뉴와 가격을 결정하며, 실시간 영업과 시즌 랭킹까지 연결되는 흐름으로 구성했습니다.

핵심 목표

실제 도시 데이터를 게임의 의사결정 요소로 변환해 입지 선택과 운영 전략이 결과에 반영되는 경영 시뮬레이션 구현

주요 기술

Spring Boot, React, TypeScript, MySQL, Redis, HDFS, Three.js, Unity WebGL

주요 기능

3D 서울 지도, 버블 뉴스, 영업 준비와 발주, 5종 액션, 일일 리포트, 시즌 랭킹

프로젝트 개요 대시보드 지역 선택 영업 준비 / 뉴스 실시간 영업 일일 리포트 시즌 랭킹 마이페이지

프로젝트 개요

Bubble POP-UP은 서울시 실제 데이터를 바탕으로 팝업스토어 운영 전략을 체험할 수 있도록 설계한 시뮬레이션 게임입니다. 사용자는 상권을 선택하고, 메뉴와 가격을 정하고, 영업 중 액션을 실행하며, 시즌 종료 후 ROI 기반 랭킹으로 결과를 확인할 수 있습니다.

🏪 시즌 상태와 참가 현황을 보여주는 메인 대시보드
🗺️ 홍대, 여의도, 명동, 이태원, 서울숲/성수, 신도림, 강남, 잠실 기반 지역 선택
📰 유동인구 순위와 뉴스 이벤트를 결합한 버블 뉴스
📦 영업일별 메뉴, 가격, 수량을 설정하는 준비 및 발주 시스템
🎮 할인, 긴급발주, 홍보, 나눔, 팝업이전 5종 액션이 포함된 실시간 영업 화면
📊 매출, 비용, 평판, 재고 흐름을 확인하는 일일 리포트와 시즌 랭킹
